# Casino Admin Panel

Admin panel for managing the casino platform. Built with React, Vite, Tailwind CSS, and React Router.

## Location

- **Path:** `D:\ReactProjects\casino-admin-panel`

## Setup

```bash
cd D:\ReactProjects\casino-admin-panel
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

## Features

- **Dashboard** – Overview with stats, recent activity, quick actions
- **Users** – User list and management
- **Games** – Casino games list and actions
- **Transactions** – Deposit, withdrawal, bet, win history
- **Reports** – Revenue, growth, game performance, withdrawals
- **Settings** – Site name, support email, maintenance mode

## Tech Stack

- React 19 + Vite 7
- React Router DOM
- Tailwind CSS v4
- Dark theme with amber accents
